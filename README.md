# EventsAPP
This example is a Full Stack application developed using:
* Front-End : Angular
* Back-End : Expressjs
* Database : SQL SERVER

## How to run
Run frontend
`ng s`
or 
`ng serve`

Run backend
if you want to run locally run
`npm run start-l`
if you want run remotly run
`npm run start-r`

remember to create .env.local and .env.remote
